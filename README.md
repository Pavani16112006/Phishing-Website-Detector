# Overview

The project is developed using HTML, CSS, JavaScript,Python and JSON, and is implemented as a lightweight Chrome extension that helps users browse the web more safely by detecting phishing attempts before sensitive information is shared.

## Feature

Real-time phishing website detection
Rule-based analysis with explainable results
Integration with PhishTank phishing URL blacklist
Instant warning popup alert for detected threats
Reasoning for each detection
Lightweight Chrome extension architecture
Built using HTML, CSS, JavaScript, and JSON
